# PCB Slide Rule

This is a project to attempt to make a free slide rule out of a PCB. The PCB is made using the free software [KiCad](https://www.kicad.org/)

![Screen shot](docs/shots.png)

## Reference Scales

The [SVG side rule rulers](docs/) where obtained from here: https://sliderulemuseum.com/SR_Scales.shtml and converted to svgs using an [online GIF to SVG online converter](https://ezgif.com/gif-to-svg/).
